## Core SIM Datasets

### inspection — `dntt-gqwq` (~398K rows, updates daily)

| Column | Type | Notes |
|---|---|---|
| `objectid` | NUMBER (PK) | Unique inspection ID |
| `borough` | TEXT | MN / BX / BK / QN / SI |
| `status` | TEXT | OPEN / CLOSED / PENDING / DISMISSED |
| `inspection_date` | DATE | ISO 8601 from Socrata |
| `house_number` | TEXT | Street address number |
| `street_name` | TEXT | Street name |
| `latitude` | NUMBER | WGS84 — ~14% null |
| `longitude` | NUMBER | WGS84 — ~14% null |
| `the_geom` | POINT | GeoJSON point |

**Grain:** One inspection event per sidewalk location.

## Common Join Patterns

```python
# Violation rate by borough
viol_by_boro = violations_df.groupby("borough")["objectid"].count()
insp_by_boro = inspection_df.groupby("borough")["objectid"].count()
rate = (viol_by_boro / insp_by_boro * 100).rename("violation_rate_pct")

# Ramp completion by borough
ramp_stats = (
    ramp_progress_df
    .groupby("borough")["status"]
    .agg(total="count", completed=lambda s: (s == "COMPLETED").sum())
    .assign(completion_rate=lambda df: df["completed"] / df["total"] * 100)
)
```

## Known Schema Issues

| Dataset | Issue | Impact |
|---|---|---|
| `inspection` | `latitude`/`longitude` ~14% null | Spatial analysis loses ~56K records |
| `ramp_locations` | Stale since 2021 — no schema changes but data is outdated | Use `ramp_progress` instead |
| `capital_blocks` | 0 rows — schema exists but dataset is empty | Do not use |
| `permit_stipulations` | API returns HTTP 403 | Cannot extract schema |
