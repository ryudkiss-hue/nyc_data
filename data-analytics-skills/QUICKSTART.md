# Quick Start Guide

Get started with Data Analytics Skills in 5 minutes.

## Your First Skill in 3 Steps

1. Pick a skill (start with programmatic-eda)
2. Trigger it by describing your task: "I need to do exploratory data analysis on my customer dataset"
3. Provide context when asked (dataset, business context, quality thresholds)

## Common Scenarios

- New dataset → programmatic-eda
- SQL review → query-validation
- User retention → cohort-analysis
- Metric documentation → semantic-model-builder

## Learning Path

Week 1: Use programmatic-eda on 3 datasets; use analysis-planning at project start
Week 2: Add query-validation, semantic-model-builder, one analysis skill
Week 3: Build a context document; chain 3–4 skills end-to-end

## Pro Tips

- Prepare a context markdown (DB type, key tables, key metrics) and share at conversation start
- Chain skills: programmatic-eda → cohort-analysis → insight-synthesis → executive-summary-generator
- Iterate progressively; don't need perfect context upfront
- Use checkpoints to course-correct mid-workflow

## Common Patterns

**Weekly Metrics Review:** programmatic-eda → business-metrics-calculator → time-series-analysis → root-cause-investigation → executive-summary-generator

**New Dataset:** programmatic-eda → schema-mapper → semantic-model-builder

**Ad-Hoc Request:** stakeholder-requirements-gathering → analysis-planning → [analysis skill] → methodology-explainer → analysis-qa-checklist
