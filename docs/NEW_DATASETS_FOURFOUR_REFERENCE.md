## Quick Command Reference

### Fetch Dataset Info
```bash
curl -s "https://data.cityofnewyork.us/api/views/{fourfour}.json" | jq '.'
```

### Fetch Sample Data (10 rows)
```bash
curl -s "https://data.cityofnewyork.us/api/views/{fourfour}/rows.json?limit=10" | jq '.'
```

### SOQL Query Example
```
https://data.cityofnewyork.us/api/views/{fourfour}.json?$query=SELECT * LIMIT 100
```

```python
from socrata_toolkit.core.client import SocrataClient, SocrataConfig
```
