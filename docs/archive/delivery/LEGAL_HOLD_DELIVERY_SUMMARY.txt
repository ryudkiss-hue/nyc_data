FILE: legal_hold_workflow.py (542 lines)
------
Implements LangGraph-based orchestration workflow:

TypedDict:
  - LegalHoldState: Workflow state container

Class:
  - LegalHoldWorkflow: Multi-node LangGraph orchestration
    * __init__(domain, fourfour, site_id, inspector_id, start_date, end_date, filter_record_types)
    * run() -> compliance_certificate_dict
    * _build_graph(): Constructs 7-node LangGraph workflow

Workflow Nodes (7 total):
  1. fetch_records: Query Socrata for records with WHERE filters
  2. classify_records: Apply LegalHoldClassifier to each record
  3. verify_audit_trails: Check audit trail completeness
  4. check_integrity: Verify data integrity checks passed
  5. route_to_claude: Conditional routing (COMPLIANT vs AT_RISK/NON_COMPLIANT)
  6. generate_certificate: Create legal hold compliance certificate
  7. aggregate: Build final report

Helper Methods:

KEY FEATURES
================================================================================

[1] 5-DIMENSION CLASSIFICATION SYSTEM
    - Record Type: INSPECTION, VIOLATION, DISMISSAL, CORRESPONDENCE, COMPLAINT, APPEAL
    - Sensitivity: PUBLIC, SENSITIVE, PROTECTED
    - Retention Requirement: STANDARD (3y), EXTENDED (7y), INDEFINITE
    - Data Integrity: Audit trail completeness, change tracking
    - Compliance Status: COMPLIANT, AT_RISK, NON_COMPLIANT

[2] RETENTION POLICIES
    INSPECTION     → STANDARD (3 years)
    VIOLATION      → EXTENDED (7 years, appeal period + statute)
    DISMISSAL      → EXTENDED (7 years, appeal period)
    APPEAL         → INDEFINITE (ongoing disputes)
    CORRESPONDENCE → INDEFINITE (legal/administrative hold)
    COMPLAINT      → EXTENDED (7 years)

[3] SENSITIVITY CLASSIFICATION
    PUBLIC     → Aggregated, no PII, safe for disclosure
    SENSITIVE  → Location/building identifiers, FOIL-protected
    PROTECTED  → Inspector/personal data, requires legal hold

[4] AUDIT TRAIL INTEGRATION
    - AuditLogger integration: All decisions logged with run_id
    - Gap detection: Missing audit entries flagged
    - Chain of custody: Unbroken change history verified
    - Immutable tracking: Change sequence preserved

[5] DATA INTEGRITY CHECKS
    - Hash verification for audit entries
    - Missing change detection
    - Timestamp consistency checks
    - Completeness validation

[6] LANGGRAPH ORCHESTRATION
    - 7-node workflow with conditional routing
    - State machine for workflow progression
    - Parallel record processing (via Socrata batch fetch)
    - Error handling and fallback modes

[7] CLAUDE INTEGRATION
    - Model: claude-haiku-4-5-20251001 (~300 tokens)
    - Analyzes high-risk records for legal defensibility
    - Identifies gaps and remediation steps
    - Graceful degradation if Claude unavailable

[8] COMPLIANCE CERTIFICATE
    - Unique certificate ID (UUID)
    - Timestamp of issuance
    - Record counts by compliance status
    - Litigation hold summary
    - Actionable recommendations
    - Audit trail reference for traceability

[9] PRODUCTION-READY
    - Comprehensive error handling
    - Structured logging via logging module
    - Type hints on all public APIs
    - Docstrings on all classes/methods
    - Graceful fallback for missing dependencies
    - LangGraph is optional dependency (not required)

USAGE EXAMPLES

EXAMPLE 1: Basic Record Classification
----
    from socrata_toolkit.analysis import (
        LegalHoldClassifier, LegalHoldMetrics, AuditTrailMetrics, RecordType
    )
    from datetime import datetime, timezone

    classifier = LegalHoldClassifier()
    metrics = LegalHoldMetrics(
        record_id="v-12345",
        dataset_key="violations",
        fourfour="6kbp-uz6m",
        created_date=datetime(2024, 1, 15, tzinfo=timezone.utc),
        last_modified=datetime(2024, 1, 20, tzinfo=timezone.utc),
        record_type=RecordType.VIOLATION,
        has_pii=False,
        has_location_data=True,
        has_sensitive_identifiers=False,
        audit_trail=AuditTrailMetrics(
            total_changes=3,
            audit_entries=3,
            creation_logged=True,
            last_update_logged=True,
            deletion_logged=False,
            chain_of_custody_complete=True,
        ),
        data_integrity_checks_passed=True,
    )

    report = classifier.classify(metrics)
    print(f"Compliance: {report.compliance_status.value}")
    print(f"Retention: {report.retention_requirement.value} ({report.retention_years}y)")
    print(f"Litigation Hold: {report.litigation_hold_active}")
