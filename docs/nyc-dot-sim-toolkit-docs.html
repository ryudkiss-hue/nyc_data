NYC DOT SIM Analyst Toolkit

Technical reference for the Sidewalk Inspection & Management data platform — Streamlit mission control, Socrata ingestion, geospatial analysis, and reporting.

Python 3.9–3.14 Streamlit 1.x 26 Datasets NYC Open Data Socrata SODA 2.x

Introduction

The NYC DOT SIM Analyst Toolkit is a local-first Streamlit application for the NYC Department of Transportation's Sidewalk Inspection & Management (SIM) unit. It ingests 26 datasets from NYC Open Data via the Socrata SODA API, provides interactive dashboards for inspection management, GIS conflict detection, compliance analytics, and produces publication-ready reports.

data_loader.py — API Reference

Core ingestion module. All Socrata calls go through here. Uses a module-level requests.Session with retry adapter (3 retries, exponential backoff on 5xx).

fetch_dataset(dataset_key, *, limit=50_000, where=None, select=None)
Primary entry point. Checks L2 Parquet disk cache first; fetches from Socrata on miss; writes cache on success. Supports delta fetch when updated_at column is detected. On network failure with stale cache available, returns stale data with df.attrs["stale"] = True.

Parameters

dataset_key: str — Registry key (e.g. "inspection")
limit: int — Max rows (default 50,000)
where: str | None — SoQL $where clause; if None, uses default_where from registry
select: str | None — SoQL $select for column projection
Returns: pd.DataFrame
fetch_datasets_parallel(keys, *, limit=25_000)
Fetches multiple datasets concurrently via ThreadPoolExecutor (max 6 workers). Returns a dict of DataFrames; individual failures return empty DataFrames with warnings.
Returns: dict[str, pd.DataFrame]
check_socrata_connectivity(domain="data.cityofnewyork.us", timeout=5)
Pings the Socrata catalog endpoint to verify network access before rendering.
Returns: tuple[bool, str] — (success, message)
read_csv_chunked(file_obj, chunksize=50_000)
Reads a user-uploaded CSV in chunks to avoid OOM on files >50 MB. Safe to pass any file-like object (Streamlit UploadedFile, Path, etc.).
Returns: pd.DataFrame
keys_for_workflow(workflow_key)
Looks up dataset keys for a named workflow group ("qa", "spatial", "contract", "productivity") from config/datasets.yaml.
Returns: tuple[str, ...]
# Column projection example — only pull relevant columns
df = fetch_dataset(
    "inspection",
    limit=50_000,
    where="upper(borough) = 'MANHATTAN'",
    select="latitude,longitude,condition_score,defect_type,borough",
)

# Parallel loading of the QA workflow bundle
from app.data_loader import fetch_datasets_parallel, keys_for_workflow
keys = keys_for_workflow("qa")
frames = fetch_datasets_parallel(keys)
# → {"lot_info": df, "mappluto": df, "complaints_311": df, ...}

cache_manager.py — API Reference

Multi-level disk cache for Socrata data. Each dataset is serialized to a gzip-compressed Parquet file under CACHE_DIR (default: data/cache/, overridable via SOCRATA_CACHE_DIR env var). A JSON manifest tracks paths, row counts, fetch timestamps, and expiry.

Cache TTL by Dataset

DatasetTTLRationale
complaints_3111 hourHigh-frequency 311 complaints update continuously
inspection, street_permits4 hoursIntra-day inspection updates
violations6 hoursViolation records updated daily
mappluto, pedestrian_demand720 hours (30 d)Stable reference data
all others24 hoursDefault
write_cache(key, df) → Path
Serialize DataFrame to <key>_<timestamp>.parquet.gz. Updates manifest. Calls evict_old_cache() automatically to enforce the 1 GB cap.
read_cache(key, max_age_hours=None) → pd.DataFrame | None
Returns cached DataFrame if within TTL, else None. Uses per-dataset TTL from DATASET_TTL_HOURS unless overridden by max_age_hours.
read_stale_cache(key) → pd.DataFrame | None
Returns any cached file regardless of age. Used by fetch_dataset() for offline fallback when Socrata is unreachable.
evict_old_cache(max_bytes=1_073_741_824) → int
Deletes oldest .parquet.gz files until total directory size is under max_bytes. Returns bytes freed.
start_prefetch_scheduler(dataset_keys, interval_minutes=60)
Launches an APScheduler BackgroundScheduler that pre-warms the cache for the specified keys on a fixed interval. Stores the scheduler in st.session_state to prevent duplicates on re-renders.
from app.utils.cache_manager import write_cache, read_cache, start_prefetch_scheduler

# Manual cache interaction
df = read_cache("inspection")
if df is None:
    df = fetch_live_data()
    write_cache("inspection", df)

# Pre-warm high-traffic datasets on app start
start_prefetch_scheduler(
    ["inspection", "street_permits", "violations"],
    interval_minutes=60,
)

spatial/geodataframe.py — API Reference

GeoPandas integration for Socrata datasets. Converts the the_geom column (WKT string, GeoJSON dict, or JSON-encoded GeoJSON) to a proper GeoDataFrame. All functions require pip install geopandas shapely; otherwise they raise ImportError with an install hint.

geodataframe_from_socrata(df, geom_col="the_geom", crs="EPSG:4326", drop_null_geom=True)
Convert a Socrata DataFrame with a geometry column to a GeoDataFrame. Handles WKT (POINT(...)), GeoJSON dicts, and JSON-encoded GeoJSON strings. Logs the count of unparseable rows.
Returns: gpd.GeoDataFrame
spatial_join_socrata(left_df, right_df, *, left_geom_col="the_geom", right_geom_col="the_geom", how="inner", predicate="intersects", crs="EPSG:4326")
Spatial join between two Socrata DataFrames via gpd.sjoin(). Converts both to GeoDataFrames first. Predicate options: "intersects", "within", "contains".
Returns: gpd.GeoDataFrame
detect_conflicts_geopandas(inspections, permits, *, buffer_meters=50.0, insp_geom_col="the_geom", perm_geom_col="the_geom")
Find inspection locations within buffer_meters of active permit areas. Reprojects to EPSG:2263 (NY State Plane, feet) for accurate metric buffering, then back to WGS84.
Returns: gpd.GeoDataFrame of conflicting pairs
spatial_stats(gdf) → dict
Summary statistics: total_features, geometry_types (value counts), bounds ([minx, miny, maxx, maxy]), crs, and optionally total_area_sq_deg for polygon layers.
from socrata_toolkit.spatial.geodataframe import (
    geodataframe_from_socrata, detect_conflicts_geopandas, spatial_stats
)

# Convert inspection DataFrame to GeoDataFrame
gdf = geodataframe_from_socrata(df_inspections)
stats = spatial_stats(gdf)
# → {"total_features": 12430, "geometry_types": {"Point": 12430}, ...}

# Detect conflicts within 100m of permit areas
conflicts = detect_conflicts_geopandas(
    df_inspections, df_permits, buffer_meters=100.0
)
print(f"{len(conflicts)} spatial conflicts found")

Optional psycopg3-backed connector for organizations running NYC DOT data in a PostGIS database. Requires pip install 'psycopg[binary]'.

PostGISConfig(host, port, dbname, user, password, schema="public")
Connection configuration dataclass. Exposes a conninfo property for the psycopg connection string.
PostGISService(config).test_connection() → tuple[bool, str]
Runs SELECT PostGIS_Version(). Returns (True, "PostGIS 3.x") on success.
.query_within_circle(table, lat, lon, radius_m, geom_col="geom", limit=10_000)
Runs ST_DWithin(<geom>::geography, ST_SetSRID(ST_MakePoint(lon, lat), 4326)::geography, radius_m) for accurate meter-based distance.
Returns: SpatialQueryResult with .df and .row_count
.query_intersects(table, wkt_polygon, geom_col="geom", limit=10_000)

# CUSUM changepoint detection (from analytics_advanced.py)
def detect_cusum_changepoint(series: pd.Series) -> int | None:
    if len(series) < 4:
        return None
    mu = series.mean()
    cusum = (series - mu).cumsum()
    return int(cusum.abs().idxmax())

Posts a Block Kit message to a Slack incoming webhook. severity: "info" (green), "warning" (orange), "critical" (red). fields: dict of key→value pairs rendered as a 2-column section (max 10).

Two-step REST flow: (1) generate ArcGIS token, (2) addItem as GeoJSON type. Returns the item URL on success.

poll_311_new_complaints(domain, fourfour, app_token, since_iso, limit=100)

Fetches active DOB Job Application Filings (w9ak-ipjd) for a given NYC block/lot combination for cross-referencing inspection locations with active construction.
