# Project Analyst - Sidewalk Program (SW)

## Role Summary
Project Analyst in DOT's Sidewalk/Inspection Management division. Part of SIM Program Management & Budget Unit supporting:
- In-house pedestrian ramp upgrade crews
- High-priority construction planning & conflict analysis
- Contract tracking, budget analysis, productivity metrics

## Key Responsibilities

### Analysis & Reporting
- **Sidewalk repair location analysis** — Identify where repairs needed (condition-based)
- **Construction list creation** — Prioritize work, flag conflicts
- **GIS conflict detection** — Permit-to-inspection spatial analysis
- **Contract progress reports** — Monthly/ad-hoc: budget variance, productivity (sqft/day), completion %
- **Program metric tracking** — Construction data, contract performance, budget codes
- **Efficiency studies** — Routine analyses to improve contract velocity

### Operational Support
- **Conflict analysis** — Permit overlaps, construction windows, access constraints
- **High-priority inquiries** — Construction planning questions, ad-hoc requests from leadership
- **IFA program** — Justifications, recommendations, reports for in-house crews
- **Ramp program support** — Pedestrian ramp upgrades, curb metal protruding work

## Workload Pattern
- **Routine:** Weekly/bi-weekly metric tracking, standard reports
- **Urgent:** Ad-hoc construction inquiries, high-priority conflict analysis
- **Complex:** Multi-variate efficiency studies, program optimization

## Tools Currently Used
- GIS (ArcGIS or similar for conflict detection, mapping)
- Excel/SQL for data analysis, reporting
- Internal DOT systems for contract/budget data

## Key Output Types
- **Dashboards/Reports:** Contract progress, budget variance, productivity trends
- **Maps/Overlays:** Conflict zones, hotspot areas, construction schedules
- **Lists/Rankings:** Prioritized construction locations, conflict flags
- **Justifications:** IFA program recommendations, efficiency findings

## Automation Priorities (Inferred)
1. **Conflict detection** — Currently manual GIS; could auto-flag permit-inspection overlaps
2. **Contract dashboards** — Real-time progress, budget, productivity tracking
3. **Hotspot identification** — Auto-rank high-problem areas by defect + cost + age
4. **Report generation** — Template-driven weekly/monthly exports
5. **GIS analysis** — Automated spatial joins, buffering, intersection detection
