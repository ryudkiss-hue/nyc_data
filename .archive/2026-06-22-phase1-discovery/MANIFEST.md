# Archived: Phase-1 Discovery Cluster (2026-06-22)

These files were one-off artifacts from the earlier "dataset discovery / Phase 1"
effort. They are **superseded** by the current source-of-truth stack:

- **`pipeline/data/nyc_open_data_registry.json`** — authoritative registry (3,012
  datasets, full column schemas), refreshed via the Discovery/Catalog API.
- **`pipeline/data/DATA_CATALOG.md`** — auto-generated human catalog of the 90
  configured datasets (regenerated by `pipeline/regenerate_from_registry.py`).
- **`pipeline/config/socrata_datasets.json`** — the 90-dataset pipeline config
  (verified Socrata IDs, primary/foreign keys).

Why archived (not deleted): reversible; retained for historical traceability.
They referenced only each other (a self-contained cluster) — nothing in the app
or pipeline code imports or depends on them. Confirmed via reference grep on
2026-06-22.
