ARCHITECTURE
================================================================================

Data Flow:
  Socrata API (dntt-gqwq, 6kbp-uz6m, p4u2-3jgx)
         |
         v
   [fetch_data] -> Inspect, Violations, Dismissals
         |
         v
   [compute_metrics] -> Group by inspector, calculate metrics
         |
         v
   [classify_performance] -> Apply VelocityClassifier
         |
         v
   [generate_claude_assessment] -> Query Claude (~300 tokens)
         |
         v
   [generate_recommendations] -> Create coaching per inspector
         |
         v
   [build_report] -> Assemble JSON report with execution log


Scoring Weights (Composite Score):
  - Velocity:   35%
  - Quality:    25%
  - Accuracy:   25%
  - Efficiency: 15%


Performance Tiers:
  - HIGH:   composite_score >= 70.0
  - MEDIUM: 40.0 <= composite_score < 70.0
  - LOW:    composite_score < 40.0


Thresholds (all configurable):
  - Velocity HIGH: >= 6.0 inspections/week
  - Quality HIGH violations: >= 4.0 per inspection
  - Dismissal HIGH: > 35%
  - Reopened HIGH: > 15%
  - Days to close HIGH: <= 30 days


Anomalies:
  - SUDDEN_DROP: >50% velocity decrease
  - PLATEAU: flat velocity for 2+ weeks
  - ACCELERATING: >30% velocity increase
